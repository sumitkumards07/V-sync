# Security Policy

## Supported Versions

This project is archived and receives limited maintenance. Security fixes will still be accepted on the default branch.

## Reporting a Vulnerability

Please report vulnerabilities privately by opening a GitHub Security Advisory (preferred).

If that is not possible, open an issue with minimal details and request secure contact. Do not publish exploit details in public issues.

## Response Expectations

- Initial triage: best effort
- Fix timeline: no guaranteed SLA
- Public disclosure: after a fix is available or risk is otherwise mitigated
